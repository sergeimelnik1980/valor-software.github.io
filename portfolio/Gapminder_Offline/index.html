Gapminder Offline

Gapminder Offline

Gapminder Offline is an offline implementation of Vizabi—a freely provided by Gapminder framework for building visual data exploration tools.

Goals

1

Integrate Vizabi platform and add readers: DDF-CSV, Excel, CSV

2

Provide new data management features and provide support for UI Controls and interaction

3

Add translation and localization

4

Add an ability to work with several charts in parallel

5

Add a possibility to work on Windows, Linux, and Mac OS

6

Add an automatic system update

Solutions

1

Test-Driven Development (e2e tests)

2

DDF-CSV databases to store all data

3

Electron framework that provides an offline version of the application

4

DDF-CSV reader for DDF data reading

5

WS reader for querying data on WS by DDFQL

Outcome

Offline visualization tool for statistic data.
